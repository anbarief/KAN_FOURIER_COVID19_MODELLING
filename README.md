Simply run the ```KAN_F_training_Indonesia.py``` to do the neural network training using ADAM Optimizer. The results will show: model-fitting plot, estimated rates plot, and the dynamics of log(loss) plot. After every 20 epochs, the Python command line will show the loss value for each variable (S,I,R,D), and the total loss value.

Requirements: 

-Pytorch

-Numpy

-Pandas

-Matplotlib
